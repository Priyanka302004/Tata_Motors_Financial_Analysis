# Tata Motors Financial Performance Analysis
Department :
Finance & Analytics

# Objective :

To evaluate and visualize the financial performance of Tata Motors over multiple years using advanced data analytics techniques, providing actionable insights for strategic decision-making.

 # Tools & Technologies :

 1. Data Collection & Cleaning: Python (BeautifulSoup, Requests, Pandas, NumPy)

2. Database Management: MySQL / SQL Server, SQL queries for financial analysis

3. Data Visualization & Dashboarding: Power BI, Matplotlib, Seaborn, Plotly

4. Reporting & Documentation: Jupyter Notebook, PDF Reports, Markdown

5. Version Control & Collaboration: Git & GitHub

# Dataset:

1. Web-scraped financial data from Screener.in
 using Python

2. Cleaned and structured into income statements, balance sheets, and cash flow statements

3. Multi-year data for trend analysis

# Methodology

1. Data Collection & Cleaning: Scraped and structured raw financial data into usable formats (CSV/SQL tables).

2. Year-on-Year Analysis: Compared revenue, expenses, and profits across multiple years.

3. Profitability & Efficiency Metrics: Analyzed gross profit margin, net profit margin, ROE, ROA, and operating efficiency ratios.

4. Cash Flow Interpretation: Evaluated operating, investing, and financing cash flows to understand liquidity and debt management.

5. Visualization & Dashboarding: Created interactive Power BI dashboards and Python visualizations for easy insights.

6. SQL Analysis: Developed queries for quick extraction of financial KPIs and automated reporting.

# Key Analysis Performed

Revenue, gross profit, and net profit trend analysis

Expense structure vs profitability evaluation

Cash flow analysis: operating, investing, and financing activities

Profitability ratios and efficiency indicators

Multi-year comparative analysis for strategic insights

# Key Insights

Operating Cash Flows: Consistent increase, indicating strong core business performance

Financing Activities: Negative cash flows, highlighting repayment of debt obligations

Net Profit Margin: Slight year-on-year fluctuations; profitability affected by high expenses

Expense Management: High operational and administrative expenses impacted net profitability despite strong sales growth
